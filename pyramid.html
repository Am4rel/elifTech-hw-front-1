You can draw a pyramid in the console!

To do this you need to:

  1. Press the button below to start.
  2. Fill the prompt window with the number from 1 to 20.
  3. Open the console with pressing Ctrl + Shift + J (or Cmd + Shift + J on a Mac) key combination.
  4. If you see the error message in the console or want to try again, reload the page.
